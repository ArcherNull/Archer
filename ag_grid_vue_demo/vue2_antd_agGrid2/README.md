# vue2

### 安装使用
node版本 18.15.0
pnpm  8.6.2

```
// 清除缓存
pnpm store prune 

// 删除node_modules , 推荐使用remove-node-modules插件
remove-node-module

// 安装依赖
pnpm install 

// 运行项目
pnpm run dev

// 打包项目
pnpm run build
```

### 注意

在运行该项目前，请先将后台服务koa_server启动起来, 以便于更好地操作
